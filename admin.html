Administrador de catálogo

Arrastrá los productos para definir el orden en que se ven en "Todas las categorías". Usá el buscador o filtro para encontrarlos; el orden solo se guarda cuando no hay filtro activo.

Cargando productos...

Arrastrá las categorías para definir el orden en que aparecen. Dentro de cada una podés ordenar los productos de esa categoría específica.

Cargando...

Cargá el "precio anterior" de cada producto. Si es mayor al precio mayorista actual, el catálogo va a mostrar el descuento al cliente. Dejá vacío para no mostrar descuento.

Cargando productos...

Cargá banners para mostrar en la parte superior del catálogo. Subí la imagen a TiendaNube (u otro hosting), copiá la URL y pegala acá. El catálogo los rota automáticamente cada 5 segundos.

Sin banners cargados

Creá cupones de descuento que los clientes aplican en el carrito. Tipos: %, monto fijo o envío gratis (este último se marca en el pedido como "envío bonificado" para coordinarlo, no descuenta un monto). El descuento entra automáticamente a Gestión Nube. Acordate de tocar "Guardar cambios".

Sin cupones cargados

Análisis de margen de cada producto (solo lo ves vos, el cliente no). Probá un descuento y mirá cómo queda la ganancia y el margen, para decidir cuánto podés bajar sin perder plata. El margen es sobre el precio de venta.

Cargando productos...

Cargá un % de recargo opcional. Sin recargo, el producto se vende al costo. Con recargo, el precio se eleva proporcionalmente. Solo vos ves estos números — el cliente ve únicamente el precio final calculado.
